# Docker Que

> What is Docker?
- Docker helps developers build, ship, and run applications inside containers. These containers include everything required (code, libraries, runtime), ensuring the app works the same on local, staging, and production.

> Why do we use Docker?
- We use Docker to solve environment-related issues. It ensures that the application runs the same across all environments. It also improves scalability, speeds up deployment, and integrates well with CI/CD pipelines.

> What is a Container?
- A container is a lightweight, standalone executable unit that includes everything needed to run an application.

> What is a Docker Image?
- A Docker image is a read-only template used to create containers.

> What is a Dockerfile?
- A Dockerfile is a script containing instructions to build a Docker image.
